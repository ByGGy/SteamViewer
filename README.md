# SteamViewer

## Goal

Simple app using **node** & **electron** & **angular** to practice the js stack for desktop apps.

## Getting Started

* Install Nodejs (used v4.2.2 64bits)
* Install VisualStudioCode
* Create the solution folder
* Run **npm init** to initialize the solution
* Run **npm install electron --save-dev**
* Create the app folder
* Run **npm install angular[.\*] --save**
